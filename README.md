# 🛒 Supermercado Pirilampo

O **Supermercado Pirilampo** é um projeto desenvolvido exclusivamente em **JavaScript puro (Node.js)** com foco no treinamento e manipulação de estruturas de dados **JSON**. O objetivo principal foi consolidar conceitos de lógica de programação, filtragem, mapeamento e agregação de dados simulando o ecossistema de um sistema de mercado (como controle de estoque, carrinho de compras, cálculo de totais e notas fiscais).

---

## 🚀 O que foi treinado neste projeto

Como o projeto é focado 100% em lógica no ecossistema JavaScript, os principais pontos trabalhados foram:

* **Manipulação de JSON:** Consumo, estruturação e parse de objetos complexos e arrays simulando o banco de dados de produtos e vendas do supermercado.
* **Métodos Avançados de Array:** Uso intensivo de métodos nativos do JavaScript como `.map()`, `.filter()`, `.reduce()`, `.find()` e `.forEach()` para processar os itens e compras.
* **Lógica de Negócio:** Desenvolvimento de algoritmos para aplicação de descontos, soma de valores com tratamento de pontos flutuantes, verificação de estoque e relatórios de faturamento.

---

## 🛠️ Tecnologias Utilizadas

* **JavaScript (ES6+):** Linguagem utilizada para toda a construção da lógica de programação.
* **Node.js:** Ambiente de execução para rodar os scripts JavaScript diretamente no terminal.
